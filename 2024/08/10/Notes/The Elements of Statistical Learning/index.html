# Mathematical Formulas

1. Scalar Derivative with respect to a Vector

∂(aTx)∂x=a\frac{\partial (\mathbf{a}^T \mathbf{x})}{\partial \mathbf{x}} = \mathbf{a} ∂x∂(aTx)​=a

2. Quadratic Form Derivative with respect to a Vector

∂(xTAx)∂x=(A+AT)x\frac{\partial (\mathbf{x}^T \mathbf{A} \mathbf{x})}{\partial \mathbf{x}} = (\mathbf{A} + \mathbf{A}^T) \mathbf{x} ∂x∂(xTAx)​=(A+AT)x

3. Linear Form Derivative with respect to a Vector

∂(bTx)∂x=b\frac{\partial (\mathbf{b}^T \mathbf{x})}{\partial \mathbf{x}} = \mathbf{b} ∂x∂(bTx)​=b

4. Trace Derivative with respect to a Matrix

∂tr(AX)∂X=AT\frac{\partial \mathrm{tr}(\mathbf{A}\mathbf{X})}{\partial \mathbf{X}} = \mathbf{A}^T ∂X∂tr(AX)​=AT

5. Quadratic Form Derivative with respect to a Matrix

∂(xTAx)∂A=xxT\frac{\partial (\mathbf{x}^T \mathbf{A} \mathbf{x})}{\partial \mathbf{A}} = \mathbf{x} \mathbf{x}^T ∂A∂(xTAx)​=xxT

6. Frobenius Norm Derivative

∂∥X∥F2∂X=2X\frac{\partial \|\mathbf{X}\|_F^2}{\partial \mathbf{X}} = 2\mathbf{X} ∂X∂∥X∥F2​​=2X

7. Diagonal Matrix Derivative with respect to a Vector

∂(xTdiag(a)x)∂x=2diag(a)x\frac{\partial (\mathbf{x}^T \mathrm{diag}(\mathbf{a}) \mathbf{x})}{\partial \mathbf{x}} = 2 \mathrm{diag}(\mathbf{a}) \mathbf{x} ∂x∂(xTdiag(a)x)​=2diag(a)x

8. Matrix Product Derivative with respect to a Matrix

∂(XA)∂X=AT\frac{\partial (\mathbf{X}\mathbf{A})}{\partial \mathbf{X}} = \mathbf{A}^T ∂X∂(XA)​=AT

9. Cross-Entropy Loss Derivative with respect to the Input

∂Cross Entropy(y,y^)∂y^=−yy^+1−y1−y^\frac{\partial \text{Cross Entropy}(y, \hat{y})}{\partial \hat{y}} = -\frac{y}{\hat{y}} + \frac{1 - y}{1 - \hat{y}} ∂y^​∂Cross Entropy(y,y^​)​=−y^​y​+1−y^​1−y​

10. Softmax Derivative with respect to the Input

∂σ(z)i∂zj=σ(z)i(δij−σ(z)j)\frac{\partial \sigma(\mathbf{z})_i}{\partial z_j} = \sigma(\mathbf{z})i (\delta{ij} - \sigma(\mathbf{z})_j) ∂zj​∂σ(z)i​​=σ(z)i(δij−σ(z)j​)
